# 🧠 ContextAI - RAG Chatbot

A Retrieval-Augmented Generation (RAG) chatbot built using LangChain, Ollama, and ChromaDB.

---

## 🚀 Features

- Local LLM using Ollama
- Vector database using ChromaDB
- Semantic search using embeddings
- Streamlit UI

---

## ✨ Dynamic File Upload Feature

- Upload your own `.txt` file
- Ask questions dynamically
- No fixed dataset required

---

## 🛠 Tech Stack

Python | LangChain | Ollama | ChromaDB | Streamlit

---

## ▶️ Run Locally

```bash
conda activate rag_env
python -m streamlit run app.py
